# Vietnam Premium Birdnest - Next.js Project
**GitHub Repo**: [https://github.com/Alex707ddd/birdnest_vn.git](https://github.com/Alex707ddd/birdnest_vn.git)

## Project Overview
This is a high-end commercial website for Vietnam Birdnest, rebuilt using Next.js 15, Tailwind CSS, and Framer Motion for a "Liquid Glass" premium aesthetic.

## Features
- **Commercial WhatsApp CRM**: Booking form directly triggers WhatsApp consultations.
- **i18n Support**: Full support for Traditional Chinese, Simplified Chinese, English, and Vietnamese.
- **Performance Optimized**: Using Next.js Image optimization and optimized video assets.
- **Responsive Design**: Pixel-perfect layout from mobile (375px) to ultra-wide desktop.

## Developer Notes
- Framework: Next.js (App Router)
- Styling: Vanilla CSS + Tailwind
- Animation: Framer Motion
