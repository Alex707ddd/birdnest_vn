## 1. 核心目標與定位 (Executive Summary)
- **品牌形象**: 「傳統精美 + 專業可信」。以高品質圖像/影片為核心，建立真材實料的第一印象。
- **教育價值**: 推廣燕窩作為「東方養生珍品」的價值 (蛋白質與 EGF)，並解決消費者對品質、種類與燉煮的疑慮。
- **目標市場**: 支援 **繁體中文 (TC)**、**簡體中文 (SC)**、**英文 (EN)**、**越南文 (VN)**。

## 3. 視覺與開發策略 (Design & Development)

### 3.1 視覺系統 (Traditional Exquisite Style)
- **色彩**: 暖玉白 (`#F5F5F0`)、宮廷金 (`#C5A065`)、墨棕 (`#4A3B32`)。
- **字體**: 中文標題用 `Noto Serif TC` (典雅)；內文用 `Noto Sans TC` (清晰)。
- **材質**: 背景疊加輕微宣紙/絲綢紋理，區塊使用細金線邊框。

### 3.2 多語言架構 (Multilingual)
- **框架**: Next.js + Sub-path Routing (`/vn/`, `/en/` 等)。
- **UI**: 頂部導航列包含 `繁 / 简 / EN / VN` 文字切換器。

## 4. 燕窩百科與教學 (Education & Tutorial)

### 4.1 燕窩種類圖鑑
| 種類 | 結構特徵 | 燉後特性 | 市場定位 |
| :--- | :--- | :--- | :--- |
| **碎燕** | 短細纖維，無長絲 | 膨脹率高，柔軟細滑 | 經濟型，適合大量銷售 |
| **混合燕** | 外層長絲，內芯短碎 | 均勻膨脹，帶有嚼感 | 中價位，市場最暢銷 |
| **頂級長絲燕** | 全體粗長均勻纖維 | 絲條完整，軟中帶微彈 | 高端贈禮，頂級品質 |

### 4.2 燉煮五部曲 (The 5-Step Ritual)
1. **浸泡 (Soak)**: 
   - 碎燕: 20-30 min
   - 混合燕: 30-45 min
   - 頂級長絲: 45-60 min
   *註：必須使用室溫水，禁用熱水。*
2. **清理 (Clean)**: 檢查雜質，必要時以鑷子清除，輕洗瀝乾，保留營養。
3. **隔水燉煮 (Double-Boil)**: 放入陶瓷/玻璃碗，加水剛好覆蓋，小火燉煮 (25-60 min 視種類而定)。
4. **添加配料 (Infuse)**: 最後 5-10 分鐘加入冰糖、紅棗、蓮子或枸杞。
5. **成品判斷 (Serve)**: 成品應完全膨脹、絲條清晰且具天然淡甜味。最佳保存：冷藏最多 3 天。

### 4.3 核心禁忌 (Secret Tips)
- **不可高溫**：高溫會破壞蛋白結構。
- **不可直接煮沸**：必須小火隔水燉。
- **不可過度清洗**：避免流失營養。

## 5. 頁面資訊架構 (Information Architecture)
1. **首頁 (Landing)**: Hero 影片 -> 信任證據 -> 精選產品 -> 工序時間軸。
2. **產品詳情 (PDP)**: 微距圖庫 -> 放大細節 -> 食用指南分頁。
3. **品牌/工藝頁**: 採摘到包裝的真實短片紀錄。
4. **百科教學頁**: 下拉式種類對比與動態燉煮教學。
