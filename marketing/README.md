# 燕窩生意 Marketing 資料夾說明

本資料夾包含了針對你的越南燕窩生意在香港及台灣市場的詳細分析與策略建議。

## 📄 檔案列表

### 1. [通用商業思維模型庫 (GENERAL_BUSINESS_FRAMEWORKS.md)](./GENERAL_BUSINESS_FRAMEWORKS.md) 🧠 **(知識資產 - 可複製到其他生意)**
*   **內容:** 提款機模式 (Cash Machine)、螞蟻王模式 (Ant King)、科技合夥人模式 (Tech Partner)。
*   **用途:** 這是你從這門生意中學到的**「底層邏輯」**，以後做任何生意都能用。

### 2. [通用技術資產庫 (TECHNICAL_ASSETS_TEMPLATE.md)](./TECHNICAL_ASSETS_TEMPLATE.md) 🛠️ **(技術指標 - 復用建站/AI設定)**
*   **內容:** 快速建站 SOP、MCP Skill 配置、掛載指令。
*   **用途:** 像 MetaTrader 的 Indicator 一樣，讓 AI 快速執行技術任務。

### 3. [給少東家 (2x歲女當家) 的提案策略 (supplier_pitch.md)](./supplier_pitch.md) 🤵‍♀️ **(談判必讀)**
*   **對象:** 越南工廠的年輕接班人。
*   **策略:** 你不是「倒爺」，你是幫她品牌國際化的 **「AI 科技合夥人」**。
*   **方法:** 用「AI 燕窩管家」換取獨家代理權與最優批發價。

### 3. [高量低頻策略報告 (high_volume_strategy.md)](./high_volume_strategy.md) 🌟 **(最重要 - 你的 Cash Machine)**
*   **內容:** 專為「股票交易員」設計的模式。
*   **香港:** 海味街回收商「進化路徑」(不用飛的遠端交易法)。
*   **台灣:** **Uncle 合作模式 (螞蟻王)** 詳解。
*   **財務:** 真實 ROI 試算 (11% - 42%)。

### 4. [回收商通訊錄/黑皮書 (contact_list.md)](./contact_list.md) 📞 **(實戰工具)**
*   **內容:** 香港與台灣的回收商 WhatsApp / 電話名單。

### 5. [可行性評估 (feasibility_check.md)](./feasibility_check.md)
*   **內容:**
    *   **物流升級:** 「代收 + 親飛交割」混合模式。
    *   **法規更新:** B2B 回收不用貼標籤，但需有越南衛生證明 (Health Cert)。

### 6. [市場調查數據報告 (market_research.md)](./market_research.md)
*   **內容:** 成本與利潤分析、競品價格、進出口法規。

### 7. [B2B 策略報告 (b2b_strategy.md)](./b2b_strategy.md)
*   **內容:** 包含針對 "Whale Clients" (鯨魚散客) 的大戶開發策略。

## 🚀 最終執行建議 (Executive Action)
*   **第一步:** 參考 `contact_list.md`，傳樣品照給海味街回收商問價。
*   **第二步:** 與越南女當家談判 (用 `supplier_pitch.md` 的 AI 概念)，拿到散裝最低價。
*   **第三步:** 越南發貨香港代收 -> 您飛去收錢 -> 獲利離場。
